Work

Personal Portfolio

To refresh my knowledge of HTML & CSS and to practice my newly-learned Javascript and JQuery skills, I created a personal portfolio that put these abilities to use. I wanted to write as much code from scratch as possible, so the responsive layout was coded sans framework. This was also the case with the HTML contact form, which is validated with Javascript. Another feature of the site is the Javascript modal (this!) that opens when samples of my work are clicked. This was the biggest challenge of making the site, and through the process I learned an important principle of not nesting functions in loops! I also learned how to incorporate the JQuery plugins sticky nav and scroll-to-top. (Finished Aug. 2015)

Asian Christian Fellowship (ACF)

"Flipping" the Wash U student organization ACF's website was a project that I planned and created entirely from start to finish. I used the Bootstrap framework to create the elegant one-page layout and responsive design. The most difficult part about creating this website was targeting specific CSS class selectors. Through this exercise, I better understood the cascading nature of CSS stylesheets. The website now includes features such as a map, accordion, and carousels. Compelling calls-to-action such as a contact form and mailing list button complete the site.(Finished Spring 2015)
